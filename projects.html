(02) Work

Selected work.

A short, honest list. Things I'm building in the open — mostly in Rust, mostly at the seam between developer tools and the systems they sit on top of.

  • 2025 — now gm-crypto-rs Rust · Cryptography · no_std

    Pure-Rust SM2 / SM3 / SM4 SDK for the Chinese national crypto standards (GB/T 32905 / 32918 / 32907). The differentiator is a constant-time-designed implementation guarded by an in-CI dudect-bencher leak-regression harness — twelve gates at |τ| < 0.20. Published on crates.io as gmcrypto-core.

  • 2025 — now RepoLens Rust · MCP · Agent tooling

    A repository memory and context layer for AI coding agents. Scans a repo into structured "packs," exposes them through an MCP server (26 tools), and pairs them with a typed, exponentially-decaying memory graph so agents don't start each session cold. The goal isn't to write code for you — it's to give the agent a durable mental model of the project.

  • 2026 ghrunners Rust · CLI · macOS

    A read-only observability CLI for GitHub Actions self-hosted runners on macOS. Surfaces launchd domain, PID, process tree, log tails, and (optionally) GitHub API status in one invocation — built after an hour of debugging a second runner install on my Mac Studio convinced me the tool I wanted didn't exist. Design and plan shipped; implementation in progress.
