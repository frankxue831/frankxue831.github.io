(02) 作品

精选 作品。

一份精简的清单,刻意保持简短。等到能诚实地写出来,再加新条目—— 大部分用 Rust,大部分在开发者工具与底层系统之间。

  • 2025 — 至今 gm-crypto-rs Rust · 密码学 · no_std

    国密 SM2 / SM3 / SM4 的纯 Rust SDK (GB/T 32905 / 32918 / 32907)。 核心差异点是常量时间设计的实现, 由 CI 中的 dudect-bencher 泄漏回归门控守护—— sm4-bitsliced-simd 特性下 15 个目标, |τ| < 0.20。 当前发布 v0.6.0(2026-05-14), v0.7.0 密码模式开发中;下一站 v0.8 AEAD(SM4-GCM + SM4-CCM)。 已发布到 crates.io, 名为 gmcrypto-core。 继续阅读 →

  • 2025 — 至今 RepoLens Rust · MCP · Agent 工具

    为 AI 编程 Agent 提供的仓库记忆与上下文层。 把仓库扫描成结构化的"包", 通过 MCP server(26 个工具)暴露, 再配上一个类型化、按指数衰减的记忆图谱, 让 Agent 不必每次都从零开始。 目标不是替你写代码——而是给 Agent 一份对项目持久的心智模型。

  • 2026 ghrunners Rust · CLI · macOS

    macOS 上 GitHub Actions self-hosted runner 的只读观测 CLI。 一次调用便能展示 launchd domain、PID、进程树、日志尾、 (可选)GitHub API 状态—— 起因是我花了一个小时调试 Mac Studio 上的第二个 runner 安装, 确认我想要的工具并不存在。 设计文档与实施计划已就绪,实现进行中。
