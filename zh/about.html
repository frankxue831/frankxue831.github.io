关于我

简单介绍一下我最近在做什么，以及我在意什么。

我叫 Frank，是软件工程师。主要写 Rust，做的多是开发者工具，以及这些工具所依赖的底层系统。

最近主要在做三件事。第一件是 gm-crypto-rs，纯 Rust 的国密 SM2 / SM3 / SM4 SDK，重点是让处理密钥、敏感数据的代码路径按常量时间实现，并在 CI 里用 dudect 做侧信道泄漏检测。第二件是 RepoLens，给 AI 编程 Agent 用的仓库上下文层，想让新的会话不必每次都从零开始理解项目。第三件是 ghrunners，macOS 上的一个小工具，用来查看 GitHub Actions self-hosted runner 的运行状态，把 launchd、进程、日志和 GitHub API 里的信息放在一起看。

这些项目方向不完全一样，但我关心的东西比较一致：命名要清楚，状态要讲明白；不确定的地方不要装确定，出问题前最好有检查能先拦一下。代码里的注释也应该解释“为什么”，而不只是重复“做了什么”。

这个网站会慢慢补起来。想看我最近在做什么，GitHub 最直接。
