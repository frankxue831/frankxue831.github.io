关于, 简单说说。

一份草稿。说得少一些但诚实一些,比说得多但含糊要好。

我是 Frank — 一名软件工程师,主要写 Rust, 主要工作在开发者工具与底层系统之间的接缝。

最近做的三件事:用纯 Rust 实现国密 SM2 / SM3 / SM4 的 SDK, 关键路径常量时间设计、并在 CI 中用 dudect 守护; 为 AI 编程 Agent 做一个仓库记忆层,带类型化、按指数衰减的记忆图谱, 通过 MCP 暴露;以及一个小型 macOS CLI, 只是为了让"调试 self-hosted runner"这件事不再要花一小时。

把它们串起来的,是对工作里不那么"亮眼"的部分的偏爱—— 清晰的命名、能在发布前抓到泄漏的回归门控、 终于把"为什么"解释清楚的那行注释。有立场的那些东西。
