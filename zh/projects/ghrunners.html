ghrunners

macOS 上 GitHub Actions self-hosted runner 的一次性、只读观测 CLI。

{% include project-summary.html %}

是什么

ghrunners 会检查本机 Mac 上的 GitHub Actions self-hosted runner 安装、launchd 状态、进程状态、日志, 以及可选的 GitHub API 状态。输出可以是安静的表格, 也可以是 JSON;异常会落成类型化 findings。

当前本地快照

本地/私有状态:local tag v0.1.1。当前 CLI 包括 status、describe 和 logs;GitHub API 增强通过 --api 显式开启。

跟别的工具差在哪儿

它刻意只读,也刻意允许部分输出。权限不足的路径会变成单行 finding, 而不是让整个命令失败;缺 sudo 或缺 GitHub API 权限也会体现在来源信息里, 不会被藏起来。

下一步

等仓库可以公开访问以后再加源码链接。start、stop、restart 这类控制命令不属于当前本地/私有范围。

它不是什么

  • 不是 runner 安装或注销工具。
  • 不是 daemon 或持续监控器。
  • 不支持 Linux 或 Windows。
  • 不是 fleet 管理工具。
  • 在仓库可公开访问前,它不是公开源码项目。
